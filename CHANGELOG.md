# Changelog

## 0.1.0

- Added calendar-first record entry with direct save and optional AI polish.
- Added first-run onboarding that guides new users into their first record.
- Added weekly review prompt on the calendar page.
- Added report generation with current week, previous week, recent 7 days, current month, all-record, and custom date ranges.
- Added report kinds and material toggles for weekly reports, project reviews, stage summaries, and daily summaries.
- Added project filtering for reports.
- Moved custom report templates into advanced settings so report output appears first.
- Added record search.
- Added customizable report templates.
- Added common project chips for faster record entry.
- Added project dashboard with per-project timeline, todos, risks, active status, and report shortcut.
- Added project rename and archive controls.
- Added AI provider presets, including DeepSeek.
- Added model connection testing from settings.
- Hid model endpoint behind advanced settings for preset providers.
- Added conversion from goals, ideas, and wishlist items into records.
- Added report template materials for active goals, ideas, and wishlist items.
- Added report material summary across calendar, report, and wishlist pages.
- Added goals, ideas, and wishlist sections.
- Added local JSON import/export.
- Added optional Cloudflare D1 sync with email/password login.
- Added explicit in-app data boundary copy.
- Kept model API keys local-only.
- Removed password recovery until a trustworthy email flow is available.
- Added PWA manifest, service worker, and generated app icons.
- Added architecture, examples, open-source guide, and pull request template.
