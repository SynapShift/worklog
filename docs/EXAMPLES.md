# Examples

These examples show the intended WorkLog loop: write lightly, keep structure, then generate a usable report.

## Record Fragments

```text
#work 推荐问句 完成订阅卡片验收，确认关键链路符合预期
#todo 上线效果 跟进推荐问句放量后的点击率和转化
#risk AB 实验 服务端暂不支持当前实验配置，可能影响灰度节奏
#life 运动 晚上跑步 3 公里，状态恢复了一点
```

## Structured Records

| Type | Project | Polished content |
| --- | --- | --- |
| work | 推荐问句 | 完成推荐问句订阅卡片验收，确认关键链路和交付结果符合预期。 |
| todo | 上线效果 | 持续跟进推荐问句放量后的点击率和转化表现。 |
| risk | AB 实验 | 当前服务端 AB 实验能力暂不支持目标配置，可能影响灰度节奏。 |
| life | 运动 | 晚上完成 3 公里跑步，状态有所恢复。 |

## Project Dashboard

The project dashboard groups records by project name and surfaces:

- Recent progress.
- Follow-up items.
- Risks.
- Related active goals.
- A shortcut to generate a project report.

## Report Draft

```markdown
# 本周报告

时间范围：2026-06-15 - 2026-06-21
项目范围：全部项目
记录数量：4

## 完成事项
- 完成推荐问句订阅卡片验收，确认关键链路和交付结果符合预期。

## 问题风险
- 当前服务端 AB 实验能力暂不支持目标配置，可能影响灰度节奏。

## 生活记录
- 晚上完成 3 公里跑步，状态有所恢复。

## 后续计划
- 持续跟进推荐问句放量后的点击率和转化表现。
```

## Suggested Demo Script

1. Open the calendar page.
2. Tap the plus button.
3. Add one `work` record and one `todo` record.
4. Open the project page and select the generated project.
5. Generate a project report.
6. Copy the report.
